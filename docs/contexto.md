# Proyecto: Coworking Sinergia – Sistema de Reservas

Este proyecto implementa una aplicación completa (cliente + administrador) para gestionar espacios de coworking, reservas, franjas horarias, usuarios y disponibilidad.

Incluye:

- Frontend en **React + Vite**
- Backend en **Node.js + Express**
- ORM con **Prisma**
- Base de datos **SQLite**
- Autenticación con **JWT**
- Roles: **Admin / Cliente**

El objetivo final es que un usuario pueda:

- Registrarse
- Iniciar sesión
- Ver sus próximas reservas
- Consultar el calendario de disponibilidad
- Crear nuevas reservas
- Cancelarlas

Y que un administrador pueda:

- Gestionar usuarios
- Gestionar espacios
- Bloquear franjas
- Ver calendarios por espacio
- Exportar reservas

El proyecto se encuentra en un estado avanzado, con frontend y backend conectados y múltiples pantallas funcionando.
