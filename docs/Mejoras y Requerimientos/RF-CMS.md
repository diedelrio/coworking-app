# RF-CMS-01 — Gestión de Contenido Público

## Descripción
Permitir administrar contenido dinámico visible en la landing pública.

## Actores
- Administrador

## Alcance
Incluye edición y publicación.
No incluye SEO avanzado.

## Criterios de aceptación
- Contenido publicado se expone vía API pública

## Impacto técnico
- Backend: cms
- Frontend: adminCMS
- DB: PublicContent
